Model
=====

The model is the data component of the webapp. It handles storing & loading data from the database using model classes built using SQLAlchemy, featuring convenient abstraction of database engines and relations, and facilitating queries to much more object-oriented function calls.
Generally, the model will not do anything on its own and only gets called by controller & AJAX end points to fetch or update data.
